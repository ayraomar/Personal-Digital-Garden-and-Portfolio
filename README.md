# ♡ Ayra's Portfolio ♡

Welcome to my personal digital garden! This is a space where I document my journey as a **Girl in STEM**, share my university projects, and archive my programming "crashouts."

## ✨ Unique Features
- **The Diary:** A "tweet-style" dev log for real-time progress.
- **Project Case Studies:** Detailed breakdowns of my work in AI, SDA, DSA, and OOP.
- **Custom Aesthetic:** Built with a typewriter font and a "kawaii-indie" vibe because tech doesn't have to be boring.

## 🛠️ Built With
- HTML5 / CSS3 / JavaScript
- Custom hosted fonts (OldNewspaper)
- Netlify for deployment
- A template from PetraPixel (heavily customized!)
